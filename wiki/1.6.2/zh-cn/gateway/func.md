


# 方法

此处列出所有 Gateway 的方法

- provider

  Gateway 基于 Repeater, 此处的 Provider 将转交给 Gateway
 
- log

  Gateway 所使用的 Log 组件

- capture

  转交于 Repeater capture

- ssl

  转交于 Repeater ssl

- cros

  配置 Gateway 支持跨域请求

- config

  转交于 Repeater config

- auth

  请求授权验证
  
- noauth

  不进行授权的请求配置

- mapping

  网关映射
  
- listen
  监听
