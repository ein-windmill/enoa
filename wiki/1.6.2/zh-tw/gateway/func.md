


# 方法

此處列出所有 Gateway 的方法

- provider

  Gateway 基於 Repeater, 此處的 Provider 將轉交給 Gateway
 
- log

  Gateway 所使用的 Log 元件

- capture

  轉交於 Repeater capture

- ssl

  轉交於 Repeater ssl

- cros

  配置 Gateway 支援跨域請求

- config

  轉交於 Repeater config

- auth

  請求授權驗證
  
- noauth

  不進行授權的請求配置

- mapping

  閘道器對映
  
- listen
  監聽
